#System aukcyjny najprościej jak się da.


##Uruchomienie
###1. Wymagania wstępne:

* System z rodziny Unix
* Ruby 1.9.x
* Bundler


###2. Instalacja:


    $ cd micro_ebuy

    $ bundle install

    $ bundle exec rake db:create

    $ bundle exec rake db:schema:load

    $ rails s
